# Movies-Gallery-API
This is an Open API for Movies, created with ASP.NET Core Web API.

Technologies and Frameworks :
 > ASP.NET Core,
 > EF Core (Entity Framework Core),
 > Swagger

NOTE : Change configuration settings and add migration before running the project :)
